== Installation ==

Installing this plugin as regular wordpress plugin. First you need to install Visual Composer plugin then install Hover Effects For Visual Composer.

This addon plugin works like other page builder addon plugin. Go to new page then click on Visual Composer builder then click on Add Element you should see Hover Effects menu as Name "Hover Effects" on Menu Top, click on it you will see Hover Effects then click on it and browse an image and customize according to your needs.
